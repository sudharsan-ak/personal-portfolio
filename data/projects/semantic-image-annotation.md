# Project: Semantic Understanding of an Image using Multi-Label Image Annotation

## Overview
A MATLAB-based computer vision project that performs semantic understanding of images using multi-label image annotation techniques.

## Technologies
MATLAB, Computer Vision, Image Processing, Multi-Label Classification

## Key Features & Achievements
- Built a MATLAB-based computer vision pipeline for semantic image understanding
- Implements multi-label image annotation to identify multiple objects/concepts in a single image
- Demonstrates knowledge of computer vision, image processing, and machine learning classification
