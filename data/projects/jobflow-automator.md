# Project: JobFlow Automator

## Overview
A TypeScript/Node.js CLI tool with Playwright browser automation and a human-in-the-loop workflow to automate job applications across multiple job boards, reducing manual effort by 75%.

## Technologies
TypeScript, Node.js, Playwright, CLI

## Key Features & Achievements
- Built a TypeScript/Node.js CLI tool with Playwright and a human-in-the-loop workflow
- Automates job applications across 3 job boards
- Reduces manual application effort by 75%
- Human-in-the-loop design ensures user reviews and approves each submission before it goes through
