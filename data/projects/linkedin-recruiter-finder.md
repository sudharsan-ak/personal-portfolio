# Project: LinkedIn Recruiter Finder

## Overview
A Chrome extension that scans LinkedIn job and People pages to find recruiter profiles, track outreach history, and reduce manual research time during job searches.

## Technologies
JavaScript, Chrome Extension API, HTML, CSS

## Key Features & Achievements
- Built a Chrome extension that scans LinkedIn job and People pages to find recruiter profiles
- Tracks recruiter outreach history to avoid duplicate contact
- Reduces manual outreach research effort significantly
- Uses Chrome storage API for persistent history across sessions
