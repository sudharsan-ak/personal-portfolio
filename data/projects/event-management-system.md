# Project: Event Management System

## Overview
A web application for event booking and cancellation built using Java Servlets and JSP. Automated test cases with Selenium IDE, achieving 100% statement coverage and PIT mutation coverage.

## Technologies
Java, JSP, MySQL Workbench, JUnit, Selenium, Eclipse

## Key Features & Achievements
- Built a web app for event booking and cancellation using Java Servlets and JSP
- Automated test cases with Selenium IDE
- Achieved 100% statement coverage and PIT mutation coverage
- Demonstrates advanced software testing methodologies
